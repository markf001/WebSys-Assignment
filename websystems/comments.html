Website Structure

The website is made up of 4 different HTML webpages and 1 CSS stylesheet. Each webpage is described below:

Filename Description
index.html The homepage of the website. It contains a brief introduction about myself and my current hobby
past.html Contains information about my past, including my high school and university experience
future.html Contains information about my future goals
comments.html Contains comments about the website structure and design choices
websystems.css Contains the styling for the website

Header

The header div which contains the website title "Web Systems Assignment" is utilised across all of the webpages using the h1 tag.

The header div also contains the navigation bar which consists of 4 buttons that direct to their respective webpages. The buttons are for the home page (index.html), past page (past.html), future page (future.html) and comments page (comments.html).

CSS Configurations

The table below highlights the main CSS configurations implemented:

Selector What it does Used on
body Reset margins, set base font to Fira Sans Entire site
#header Dark bar with light text behind title & navbar Header
ul.navbar Flex layout, centers nav items horizontally Navigation
ul.navbar a:hover & a.active Hover/active highlight Navigation links
h2 Section headings with bottom border underline Page section titles
img.center & video Centered media via block + auto margins Future page images & home page video
footer, .footer-left, .footer-right Dark footer, left/right text alignment Footer on all pages
table & thead th Centered table, collapsed borders, dark header row Projects/comments tables

Aesthetics

The aesthetics used for this website was chosen based on readability and how pleasing it is to read. The colour theme used is a combination of a beige background for bars, light green background and white text for headers of the bars. This theme aims to be easy on the eyes and not too harsh. The font used compliments this theme as it is a sans-serif font which is also easy to read and is modern looking, resulting in a website that is not too overbearing and is simple.

Alt tag

Though the tag is not used much in this website, it is important for accessibility. The alt attribute provides a text alternative for images, which can be helpful for users who are experiencing visual interferences.
